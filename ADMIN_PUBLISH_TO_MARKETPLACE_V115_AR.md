# v11.5 — النشر من لوحة الإدارة إلى تصفح الفرص

## الهدف

بعد أن يرسل صاحب المشروع طلبه، يمكن للإدارة مراجعته ثم نشره في صفحة تصفح الفرص دون استخدام SQL يدوي كل مرة.

## ما تمت إضافته

- زر في لوحة الإدارة عند طلبات المشاريع:
  - نشر في تصفح الفرص.
  - تحديث النشر في تصفح الفرص.
- تحويل بيانات `project_submissions` إلى سجل في `opportunities`.
- تسجيل:
  - `marketplace_published`
  - `opportunity_slug`
  - `published_at`

## الفائدة المباشرة

- صاحب المشروع يرى مشروعه يتحول إلى فرصة منشورة بعد المراجعة.
- الزائر يرى الفرصة في `/marketplace`.
- الإدارة لا تحتاج نسخ SQL يدوي لكل مشروع.

## SQL مطلوب

شغّل:

```text
supabase/admin-publish-to-marketplace-v115.sql
```

## المسار الصحيح

```text
إرسال المشروع
→ مراجعة إدارية
→ نشر في تصفح الفرص
→ ظهور في /marketplace
```
