# حراج انڤست — نسخة واجهة مطورة

هذه نسخة مطورة بصريًا من مشروع حراج انڤست، تشمل:

- الصفحة الرئيسية بشكل احترافي.
- صفحة مشاريع بفلاتر وبطاقات محسنة.
- صفحة تفاصيل مشروع أقوى بالمؤشرات المالية ونقاط القوة والمخاطر.
- نموذج إضافة مشروع أكثر تنظيمًا.
- شعار حراج انڤست داخل `public/brand`.
- نفس بنية Next.js الصحيحة: `app/`, `components/`, `package.json` في جذر المشروع.

## التشغيل المحلي

```bash
npm install
npm run dev
```

افتح:

```text
http://localhost:3000
```

## الرفع إلى GitHub Desktop

1. فك ضغط الملف.
2. ادخل مجلد `haraj-invest-upgraded-ui`.
3. تأكد أن داخله مباشرة:

```text
app
components
public
package.json
```

4. انسخ الملفات فوق الريبو المحلي.
5. Commit ثم Push.

## ملاحظة

هذه النسخة ما زالت تستخدم بيانات Mock داخل `data/mock.ts`. الربط الفعلي مع Supabase يتم في مرحلة لاحقة.
