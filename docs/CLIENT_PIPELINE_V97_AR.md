# خط متابعة العملاء — v9.7

## مراحل خط المتابعة

### 1. Lead جديد
مستخدم أرسل اهتمام أو صاحب مشروع تواصل.

### 2. مؤهل مبدئيًا
تم فهم نوع الطلب والهدف.

### 3. يحتاج معلومات
تنقصه أرقام، وصف، مستندات، أو توضيح.

### 4. خدمة مناسبة
تم تحديد خدمة محتملة:
- تجهيز ملف فرصة.
- عرض مميز.
- NDA.
- تحسين طلب اهتمام.

### 5. متابعة
تم إرسال رسالة متابعة بموعد أو خطوة محددة.

### 6. مغلق
إما اكتمل، مؤجل، أو غير مناسب.

## قاعدة متابعة

لا تترك أي محادثة دون خطوة تالية واضحة:
- أرسل البيانات.
- حدد موعد.
- اطلب NDA.
- جهز ملف فرصة.
- أعد التواصل بعد مدة.
