# دليل صياغة إخلاء المسؤولية — v9.9

## صياغة مختصرة مناسبة

حراج إنفست منصة لتنظيم عرض الفرص التجارية والمشاريع وتسهيل التواصل الأولي. لا تقدم المنصة توصيات استثمارية أو ضمانات للعوائد أو نجاح الصفقات. يجب على كل مستخدم إجراء الفحص المالي والقانوني والتشغيلي المستقل قبل اتخاذ أي قرار.

## صياغة داخل صفحة فرصة

المعلومات المعروضة أولية ومقدمة لغرض التعريف والتنظيم، ولا تعد توصية استثمارية. أي متابعة يجب أن تتم بعد مراجعة مستقلة وطلب المستندات المناسبة.

## صياغة عند NDA

طلب NDA لا يعني اعتماد الفرصة أو التوصية بها، بل هو مسار لتنظيم مشاركة المعلومات الحساسة بين الأطراف.

## صياغة عند إرسال اهتمام

إرسال الاهتمام خطوة أولية غير ملزمة، وتهدف إلى فتح تواصل منظم حول الفرصة.
