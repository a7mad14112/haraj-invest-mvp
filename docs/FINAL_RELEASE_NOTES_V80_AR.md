# ملاحظات الإصدار النهائي — v8.0

## الهدف

تحسين الأداء، SEO، واستقرار المسارات قبل اعتماد النسخة كإصدار مستقر.

## ما تم تحسينه

- تحديث sitemap.
- تحديث robots.
- تقوية metadata العامة.
- تحسين حالات الأداء عبر CSS.
- دعم prefers-reduced-motion.
- تحسين استقرار الجوال.
- تنظيف وقائي للمسارات المرفوضة.

## لا يوجد

- لا SQL جديد.
- لا تغيير على Supabase.
- لا تغيير على Admin.
- لا `/discover`.
- لا `/compare`.
