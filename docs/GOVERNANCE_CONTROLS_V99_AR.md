# ضوابط الحوكمة — v9.9

## الهدف

حماية المنصة والمستخدمين من سوء الفهم، الوعود غير الدقيقة، أو اعتبار المنصة جهة توصية استثمارية.

## مبادئ أساسية

1. المنصة تنظم عرض الفرص ولا تضمن نجاحها.
2. المنصة لا تقدم توصيات استثمارية.
3. أي قرار يحتاج فحصًا مستقلًا.
4. لا يتم عرض وعود عائد كضمان.
5. لا يتم اعتبار البيانات المنشورة بديلًا عن القوائم أو المستندات الأصلية.
6. NDA لا يعني تأكيد جودة الفرصة، بل تنظيم مشاركة المعلومات الحساسة.

## لغة يجب تجنبها

- فرصة مضمونة.
- عائد مضمون.
- استثمار آمن.
- صفقة مؤكدة.
- فرصة لا تفوّت بصيغة وعدية.
- تم التحقق بالكامل إذا لم يحدث تحقق كامل.

## لغة مناسبة

- فرصة معروضة بشكل منظم.
- بيانات أولية تحتاج مراجعة.
- يتطلب فحصًا مستقلًا.
- يمكن طلب NDA لمشاركة تفاصيل أعمق.
- المعلومات لا تعد توصية استثمارية.

## قاعدة

أي نص قد يُفهم كتوصية أو ضمان يجب تعديله فورًا.
