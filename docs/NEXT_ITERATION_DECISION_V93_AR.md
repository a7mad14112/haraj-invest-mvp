# قرار التطوير التالي — v9.3

## لا تطور مباشرة

راقب أولًا:

- هل تصل طلبات اهتمام؟
- هل تصل طلبات NDA؟
- هل يضيف أصحاب المشاريع بيانات كافية؟
- هل المستخدمون يفتحون صفحة الفرصة؟
- هل الجوال يسبب مشاكل؟

## سيناريوهات القرار

### 1. الزيارات موجودة لكن الطلبات قليلة

طوّر:
- صفحة تفاصيل الفرصة.
- CTA.
- الثقة والبيانات.
- جودة الفرص المعروضة.

### 2. الطلبات كثيرة لكنها غير جادة

طوّر:
- نموذج الاهتمام.
- أسئلة التأهيل.
- تصنيف القدرة المالية.
- توضيح الجدية المطلوبة.

### 3. أصحاب المشاريع لا يضيفون فرصًا

طوّر:
- صفحة إضافة المشروع.
- أمثلة تعبئة.
- خدمة تجهيز ملف فرصة.
- رسائل قيمة لصاحب المشروع.

### 4. المستخدمون يطلبون NDA كثيرًا

طوّر:
- توضيح ما قبل/بعد NDA.
- إدارة مستندات مستقبلية.
- مسار متابعة الطلبات.

### 5. مشاكل الجوال كثيرة

طوّر الجوال قبل أي شيء آخر.

## الأولوية

الأولوية دائمًا للمسار الذي يؤثر على:
- إرسال الاهتمام.
- إضافة مشروع.
- طلب NDA.
- وضوح صفحة الفرصة.
