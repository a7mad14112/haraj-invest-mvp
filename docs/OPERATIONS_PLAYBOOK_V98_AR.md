# دليل التشغيل — v9.8

## الهدف

تشغيل حراج إنفست بطريقة منظمة بعد بدء وصول الطلبات.

## أنواع الطلبات

1. إضافة مشروع.
2. إرسال اهتمام.
3. طلب NDA.
4. شراكة أو تعاون.
5. استفسار عام.

## إجراءات استقبال الطلب

### 1. تحديد النوع
اسأل: هل الطلب متعلق بمشروع، فرصة، NDA، شراكة، أم استفسار؟

### 2. تقييم الجدية
راجع:
- هل يوجد اسم واضح؟
- هل توجد رسالة مفهومة؟
- هل يوجد هدف محدد؟
- هل توجد قدرة مالية أو بيانات مشروع؟
- هل يطلب خطوة منطقية؟

### 3. تحديد الحالة
- جديد.
- قيد المراجعة.
- يحتاج استكمال.
- مؤهل.
- غير مناسب.
- مغلق.

### 4. إرسال متابعة
لا ترسل رسالة عامة. اطلب معلومة محددة أو اقترح خطوة واحدة.

## قاعدة تشغيلية

أي طلب لا يحتوي على خطوة تالية واضحة سيضيع.
