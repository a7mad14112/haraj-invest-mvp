# شرائح الشراكات المستهدفة — v9.5

## 1. مكاتب المحاسبة

القيمة:
- تنظيم الأرقام.
- مراجعة القوائم.
- تجهيز ملخص مالي للفرصة.

## 2. مكاتب المحاماة

القيمة:
- NDA.
- مراجعة العقود.
- هيكلة الشراكات.

## 3. مستشارو الأعمال

القيمة:
- تجهيز ملف العرض.
- تحليل نموذج العمل.
- تحسين pitch للفرصة.

## 4. حاضنات ومسرعات الأعمال

القيمة:
- مصدر لأصحاب مشاريع.
- دعم تطوير المشاريع.
- بناء شراكات مستقبلية.

## 5. مجتمعات رواد الأعمال

القيمة:
- جذب أصحاب مشاريع.
- اختبار الرسائل.
- جمع ملاحظات مبكرة.

## 6. المستثمرون الأفراد

القيمة:
- اختبار الطلب.
- معرفة القطاعات الأكثر جاذبية.
- تحسين طريقة عرض الفرص.

## ترتيب الأولوية

ابدأ بـ:
1. أصحاب مشاريع حقيقيين.
2. مهتمين بالفرص.
3. مستشارين ماليين/قانونيين.
4. مجتمعات رواد أعمال.
5. شراكات رسمية أكبر لاحقًا.
