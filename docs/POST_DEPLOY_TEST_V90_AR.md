# اختبار ما بعد النشر — v9.0

- الشعار يظهر بوضوح.
- أزرار المسارات الأساسية تعمل.
- لا يوجد تزاحم في الجوال.
- الفوتر مرتب.

## 2. اختبار السوق

- `/marketplace`
- الفلاتر تعمل.
- عدد النتائج يظهر.
- الحفظ والمشاركة يعملان.
- لا توجد روابط `/discover` أو `/compare`.

## 3. اختبار تفاصيل الفرصة

- `/opportunity/cafe-riyadh`
- الملخص والأرقام تظهر.
- أرسل اهتمامك يعمل.
- طلب NDA يعمل.
- شريط CTA لا يغطي المحتوى.

## 4. اختبار النماذج
