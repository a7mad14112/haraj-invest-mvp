# قائمة فحص المخاطر — v9.9

## عند استقبال فرصة

راجع:

- هل الوصف واضح؟
- هل الأرقام منطقية؟
- هل يوجد تضخيم أو وعود؟
- هل توجد مستندات داعمة؟
- هل يوجد سجل أو رخصة؟
- هل يوجد تضارب في المعلومات؟
- هل يطلب صاحب المشروع نشر معلومات حساسة؟
- هل تحتاج الفرصة NDA قبل أي تفاصيل إضافية؟

## إشارات حمراء

- وعود عائد مضمون.
- رفض تقديم أي بيانات أساسية.
- تضارب واضح بين الإيرادات والربح.
- استعجال غير مبرر.
- طلب مبالغ دون معلومات كافية.
- استخدام صور أو معلومات لا يمكن نسبتها للمشروع.
- غياب أي وسيلة تحقق من صاحب المشروع.

## القرار

### قبول مبدئي
البيانات واضحة ولا توجد إشارات حمراء.

### يحتاج استكمال
الفكرة مناسبة لكن البيانات ناقصة.

### رفض أو تأجيل
البيانات مضللة، غير منطقية، أو تحمل وعودًا غير مقبولة.
