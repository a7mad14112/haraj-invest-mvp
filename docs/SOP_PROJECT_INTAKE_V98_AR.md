# SOP — استقبال المشاريع

## عند وصول مشروع جديد

راجع العناصر التالية:

- اسم المشروع.
- القطاع.
- المدينة.
- نوع الفرصة.
- القيمة المطلوبة.
- الإيرادات.
- الربح.
- المستندات.
- هل يحتاج NDA؟
- جودة الوصف.

## قرار الفرز

### مقبول مبدئيًا
إذا كان الوصف واضحًا ويوجد حد أدنى من الأرقام.

### يحتاج استكمال
إذا كان المشروع جيدًا لكن البيانات ناقصة.

### غير مناسب
إذا كان الوصف غير مفهوم أو لا يوجد هدف واضح.

## رسالة استكمال بيانات

السلام عليكم،

شكرًا لإرسال المشروع. حتى نتمكن من عرضه أو مراجعته بشكل أفضل، نحتاج استكمال التالي:

- وصف مختصر للمشروع.
- المدينة والقطاع.
- نوع الفرصة المطلوبة.
- القيمة أو النطاق المتوقع.
- الإيرادات أو مؤشرات التشغيل إن وجدت.
- هل توجد مستندات داعمة؟

بعد استلام هذه البيانات يمكن تحديد الخطوة التالية.
