# v121 — إصلاح ترتيب نص الشريط السفلي للجوال

## المشكلة (من ملاحظة مستخدم)
الشريط السفلي على الجوال (MobileQuickActions) كان يعرض كل زر بترتيب
معكوس: الكلمة البارزة (الاسم) فوق والفعل تحت، فيُقرأ من أعلى لأسفل:
"الفرص تصفح" بدل "تصفح الفرص".

## السبب
كل زر فيه عنصران مكدّسان رأسيًا: <b> (بارز) و <span> (صغير).
كان <b>الاسم</b> أولًا ثم <span>الفعل</span> — فالقراءة الرأسية معكوسة.

## الإصلاح
عكس الترتيب في الأزرار الأربعة ليُقرأ طبيعيًا (الفعل الصغير فوق، الاسم
البارز تحت):
- تصفّح / الفرص
- أضف / مشروع
- أرسل / اهتمام
- تابع / حسابي

لا تغيير في CSS أو المنطق — فقط ترتيب العنصرين داخل كل رابط.
