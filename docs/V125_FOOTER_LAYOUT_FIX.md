# V125 — إصلاح تخطيط الفوتر (قص جانبي + فراغات عمودية)

## الأعراض (من لقطة المستخدم)
1. الأعمدة مقصوصة من اليمين (overflow) — عناوين مثل "المسارات الأساسية" و"الحساب" مقطوعة.
2. فراغات عمودية ضخمة بين الروابط، لأن أعمدة Grid تتمدد لأطول عمود (5 روابط مقابل 3).

## السبب الجذري
- كان الفوتر يستخدم CSS Grid (`grid-template-columns: repeat(4,1fr)`) من بلوكي v85 و v122 معًا.
- خلايا Grid بلا `min-width:0` تتجاوز عرض الحاوية فيحدث القص.
- صفوف Grid تتمدد لارتفاع أطول خلية، فتظهر فراغات أسفل الأعمدة القصيرة.

## الإصلاح
- تحويل حاوية الروابط `.footer-nav-v85` من Grid إلى **Flexbox**:
  - `display:flex; flex-wrap:wrap; align-items:flex-start` → الأعمدة بمحاذاة علوية (لا فراغات تمدد).
  - كل مجموعة `flex: 1 1 150px; min-width:0; max-width:220px` → لا تجاوز، التفاف نظيف.
- الروابط `inline-block` بعرض تلقائي بدل `block`/grid.
- تحييد إعلانات Grid القديمة في بلوك v85 الأساسي (إزالة `display:grid !important` و`grid-template-columns` و`min-height:38px`) حتى لا تتعارض.
- تجاوب: عمودان على ≤760px، عمود واحد على ≤420px.
