# V130 — إبراز بطاقات "ابدأ الآن" كروابط واضحة

## المشكلة
بطاقات قسم "ابدأ الآن" (تصفح الفرص / أضف مشروعك / أرسل اهتمامك) كانت بخلفية بيضاء وحدود باهتة، فلا تبدو روابط قابلة للضغط. التعديلات السابقة لإبرازها لم تكن موجودة في كود v129.

## التغيير (CSS فقط في globals.css)
- خلفية كحلية متدرّجة بدل البيضاء، بهوية حراج إنڤست.
- حدود ذهبية خفيفة تبرز عند المرور (hover).
- سهم ذهبي (←) يدل بصريًا أنها رابط، يتحرك عند المرور.
- نص أبيض للعنوان وفاتح للوصف.
- تأثيرات hover و active أوضح (رفع + ظل + تغيّر اللون).
- كتلة تجاوز في نهاية الملف لضمان الفوز في cascade دون كسر قواعد قديمة.

## ملاحظة مهمة (سبب عدم ظهور التعديلات سابقًا)
التعديلات لم تكن في كود v129 أصلًا — ليست مشكلة كاش أو نشر. أُعيد تطبيقها هنا في v130.
بعد النشر، إن استمر ظهور النسخة القديمة، امسح بيانات الموقع في المتصفح (بسبب الـ service worker الذي يخزّن الأصول).
