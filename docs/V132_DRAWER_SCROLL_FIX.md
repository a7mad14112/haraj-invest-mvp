# V132 — إصلاح القائمة الجانبية (المحتوى السفلي مغطّى)

## المشكلة
عند فتح القائمة الجانبية، الشريط السفلي المتحرك (mobile-bottom-cta-v49, z-index:9999)
كان يطفو فوق القائمة (z-index:200) ويغطّي آخر عناصرها (زرّي الدخول/حساب جديد)،
ولم يكن في القائمة تمرير للوصول إليها.

## الإصلاح (CSS فقط في globals.css)
- رفع z-index القائمة عند فتحها إلى 10050 (فوق الشريط السفلي).
- تفعيل overflow-y:auto داخل القائمة مع overscroll-behavior:contain للتمرير الكامل.
- حشوة سفلية تراعي safe-area على الأجهزة الحديثة.
- إخفاء الشريط السفلي والأزرار العائمة تلقائيًا عند فتح القائمة عبر body:has(.nav-drawer.is-open)
  (حماية مزدوجة: حتى لو لم يُدعم :has، رفع z-index كافٍ).
