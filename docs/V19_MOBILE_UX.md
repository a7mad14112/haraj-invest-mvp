# v1.9 — تحسينات الجوال وتجربة المستخدم

## ما تغيّر

### 1. قائمة جوال (Hamburger Menu)
- إضافة زر قائمة يظهر تحت 980px.
- درج جانبي (drawer) ينزلق من جهة البداية مع خلفية معتمة.
- قفل تمرير الصفحة أثناء فتح القائمة.
- يُغلق تلقائيًا عند الضغط على أي رابط.
- زر الهامبرغر يتحول إلى علامة X عند الفتح (CSS animation).

### 2. حالات تحميل على النماذج
- كل أزرار الإرسال تُعطّل أثناء المعالجة (منع الضغط المزدوج).
- ظهور "جاري الإرسال" + مؤشر دوران (spinner).
- يشمل: إضافة مشروع، اهتمام، NDA، تسجيل دخول، حساب جديد، إعادة تعيين.
- النموذج يُفرّغ فقط عند نجاح الحفظ في Supabase (لا يُفرّغ عند فتح واتساب الاحتياطي).

### 3. ترتيب السوق
- إضافة قائمة "ترتيب حسب": القيمة (أعلى/أقل)، الإيرادات، المخاطر (الأقل أولًا).
- تحليل القيم النصية ("450,000 ريال") إلى أرقام للترتيب.
- "مسح" يعيد الترتيب للافتراضي.

## لم يُلمس
- Supabase، الأمان، قاعدة البيانات، الصفحة الرئيسية، المحتوى القانوني.
